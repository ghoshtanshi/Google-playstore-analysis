# Google-playstore-analysis

**BUSINESS PROBLEM **
The Google Play Store hosts millions of applications across various categories, but only a small percentage achieve high installs and strong user ratings. The company wants to understand the factors that drive app success and user engagement. By analyzing app ratings, reviews, installs, pricing, categories, app size, and update frequency, Google can identify patterns associated with high-performing apps. These insights can help developers optimize their applications, improve user satisfaction, and increase downloads. The goal is to use data-driven recommendations to enhance app performance and maximize visibility in the Google Play Store.
